GIS 三人任务工具 — 交付版
==============================

【环境】已安装 ArcGIS Pro（含 arcpy）

【解压后首次使用，按顺序】
  1. configure_path.bat（或 配置路径.bat）→ 记录本机 Pro 的 python.exe
     （若 Pro 在默认位置，launcher 也会自动检测）
  2. install_deps.bat（或 安装依赖.bat）→ 依赖装进本文件夹 pydeps/（无需管理员）
     看到 VERIFY OK 即表示成功
  3. launcher.exe → 启动工具

【数据格式】
  ★ ArcGIS Pro 文件地理数据库：.gdb 文件夹
  ★ ArcMap 个人地理数据库：.mdb 文件
    选择后会自动导入到「尝试\_mdb_cache\*.gdb」，后续任务均使用该 GDB

【解压】
  ★ 解压到任意文件夹即可，不要多套一层（避免 文件夹\文件夹\）
  ★ 打包/发送时必须保留 app\py39 与 app\py311 下的 .pyd 文件
     （微信/QQ 可能拦截 .pyd，请用 zip 或网盘）

【环境版本】
  ★ 支持 ArcGIS Pro 3.0 及以上（自带 Python 3.9 或 3.11）
  ★ Pro 3.0–3.2 用 Python 3.9；Pro 3.3+ 用 Python 3.11，程序会自动选择

【注意】
  ★ 只能通过 launcher.exe 启动
  ★ 不要直接运行 app 目录内的 .py
  ★ 成果输出在 尝试/ 文件夹

【文件说明】
  launcher.exe     Ada 专属启动器
  bootstrap.py     程序入口（由启动器调用）
  app/             加密后的 Python 程序（含 py39 / py311 两套运行时）
  python_path.txt  运行「配置路径.bat」后自动生成
